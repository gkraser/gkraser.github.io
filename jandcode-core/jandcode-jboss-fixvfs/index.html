jandcode-jboss-fixvfs¶

Провайдер виртуальной файловой системы commons-vfs для сервера приложений JBoss.

Использование¶

Порсто поместитите этот модуль в classpath приложения, которое запускается под JBoss и commons-vfs будет отрабатывать ситуации, нужные для корректной работы Jandcode.
