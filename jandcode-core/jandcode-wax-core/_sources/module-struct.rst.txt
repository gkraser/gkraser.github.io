
Структура модуля приложения
===========================

Модуль имеет следующую структуру:

    * **MODULE.PACKAGE** - java-пакет модуля
        * **module.rt** - rt-файл с описанием модуля
        * **action** - пакет для WebAction
        * **dbdata** - пакет для автоматически загружаемых в базу данных
            * **prod** - пакет для данных в базе по умолчанию
            * **test** - пакет для тестовых данных в базе
        * **model** - пакет для модели. Тут хранятся домены, dao. Внутреняя структура
          пакета произвольная. Из этого пакета (и вложенных) все rt-файлы загружаются
          в модель *default*
        * **js** - пакет для js-классов. Являеется корнем для js-пространства имен *Jc.*
        * **page** - пакет для шаблонов страниц и css
        * **tml** - пакет для gsp-шаблонов
