Настройка главной страницы¶

Для настройки главной страницы можно сделать иньекцию файлу /js/app-ini.js, содержимое файла попадет в основной заголовок /page/header.gsp после стандартной инициализации:

<%-- file: /js/app-ini-inject.js.gsp --%>
<script type="text/javascript">
    Jc.ini.myini = 1;
    <%
        def rt = getApp().getRt()
    %>
    Jc.ini.myini2 = '${rt.getValueString("app/myini:myini2")}';
</script>

Так же можно скопировать в проект файл /page/main.gsp и изменить его по своему желанию. Однако этот способ допустим только в приложениях (не в модулях!).
