
Введение
********

**Jandcode Core** - платформа для разработки бизнес приложений,
которая включает в себя следующие основные возможности:

    * модульная архитектура приложений
    * доменная модель описания предметной области
    * работа с реляционными базами данных на низком уровне
    * создание web-интерфейса пользователя с использованием javascript и ajax
    * утилиты командной строки для поддержки жизненного цикла разработки приложения
    * поддержка unit-тестирования
