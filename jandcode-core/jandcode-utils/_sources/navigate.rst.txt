
Полезности модуля jandcode-utils
********************************

В этом модуле много чего есть. Хотя большинство из этого многообразия используется где-то
там внутри библиотеки, имеются некоторые полезные вещи и для приложений.

``jandcode.utils.INamed``
    Интерфейс для объекта с именем. Такие объекты повсеместно используются в библиотеке.
    Хранятся они обычно в списке ``jandcode.utils.ListNamed``.

``jandcode.utils.UtClass``
    Статические утилиты для классов.

``jandcode.utils.UtCnv``
    Статические утилиты конвертация. Например строк в числа, дату и т.д.

``jandcode.utils.UtDate``
    Статические утилиты для дат и времени.

``jandcode.utils.UtFile``
    Статические утилиты для файлов, имен файлов.

``jandcode.utils.UtString``
    Статические утилиты для строк.

``jandcode.utils.rt``
    Rt. Подробности :ref:`тут <rt>`.

``jandcode.utils.easyxml``
    Простая работа с xml файлами в памяти. Прочитать, записать, просканировать.
