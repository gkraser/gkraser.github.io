Зарезервированные имена доменов

Зарезервированы следующие имена доменов:

  • base - базовый домен, не имеет полей. Используется как средство расширения функциональности доменов. В частности в нем определяются bean с дополнительной функциональностью для доменов.
  • id - домен с полем id типа long. Можно использовать как базовый предок для доменов-таблиц
