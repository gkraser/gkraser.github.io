Формат файла json

Содержание
  • Пример

Особенности:

  • порядок ключей не гарантирован

Пример

{
  "a1": 1,
  "b1": {
    "c1": "2",
    "d1": [
      1,
      2
    ],
    "e1": [
      {
        "z": 1
      }
    ]
  }
}

получаем:

{
    a1: 1.0
    b1: {
        c1: 2
        d1: {
            #0: 1.0
            #1: 2.0
        }
        e1: {
            #0: {
                z: 1.0
            }
        }
    }
}
